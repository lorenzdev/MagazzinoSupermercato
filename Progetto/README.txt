Per il corretto funzionamento del programma seguire le seguenti istruzioni prima di eseguirlo.


1 - Assicurarsi che XAMPP sia attivo
2 - Assicurarsi che nel progetto sia caricata la libreria "MYSQL JDBC Driver"
2 - Caricare in phpmyadmin il database fornito in modo da avere dei dati su cui testare il programma (nomeFILE)
3 - Nel file "Server.java" cambiare la variabile "DB_URL" ed inserire l'indirizzo per raggiungere il database su phpmyadmin
4 - Nel file "Server.java" cambiare la variabile "streamResult" ed inserire il percorso dove salvare il file



PER POTER LANCIARE CORRETTAMENTE IL PROGRAMMA:

1 - Mandare in esecuzione il file "Server.java"
2 - Mandare in esecuzione il file "PClient.java"*
